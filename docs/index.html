NEODECK

SSH client firmware · LilyGo T-Deck Plus · ESP32-S3

NeoDeck turns the LilyGo T-Deck Plus into a handheld SSH terminal: stored connections, a real cursor-addressable VT100/xterm screen, an on-device Claude AI assistant, and a Matrix-styled UI. Flash it straight from this page.

Flash from your browser

Your browser can't flash over USB. You are not allowed to use this on HTTP — open the HTTPS version.
Works in Chrome or Edge on desktop (Web Serial). Plug the T-Deck in over USB-C, click Install, pick the serial port.
Heads-up: the browser installer writes a full image at 0x0, so it replaces whatever firmware is on the board (including the bmorcelli Launcher, if present). Prefer to keep the Launcher? Use the SD-card method below instead.

Two ways to run NeoDeck

A · Direct (this page)

  1. Connect the T-Deck Plus by USB-C.
  2. Click Install NeoDeck above.
  3. Select the serial port, confirm erase + install.
  4. It reboots straight into NeoDeck.

B · Via the Launcher (SD)

  1. Copy the app bin to a FAT32 SD card.
  2. Insert the SD, open SD in the bmorcelli Launcher.
  3. Select NeoDeck.bin → Install.
  4. Hold select / trackball at power-on to get back to the Launcher.
For path B use the app-only firmware.bin from a build, not the merged image.

First run

  1. On the menu open WiFi, pick your 2.4 GHz network, enter the password. (ESP32 has no 5 GHz radio.)
  2. Open Connections → New connection, fill in host / user / port and a password or key, then Save.
  3. Select the connection to open the terminal. Trackball click opens the Session menu (Ctrl-C, Tab, Esc, Ctrl-D, scrollback, AI, clear, disconnect).
  4. Optionally set an Anthropic API key in Settings for the on-device AI assistant.

Settings are stored on the SD card when present (so they survive launching in and out of NeoDeck), otherwise in internal flash.

Build it yourself

pio run                 # compile
pio run -t upload       # flash over USB-C
pio device monitor      # serial @115200

Full source, hardware notes and architecture: github.com/justynroberts/NeoDeck.

NeoDeck is a personal handheld tool — secrets are stored in plaintext on the device. Treat it accordingly.
